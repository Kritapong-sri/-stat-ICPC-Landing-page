About

เกี่ยวกับการแข่งขัน

ICPC คืออะไร?

ICPC (International Collegiate Programming Contest) คือการแข่งขันโปรแกรมมิ่งระดับนานาชาติสำหรับนิสิตนักศึกษา ที่เก่าแก่และมีชื่อเสียงที่สุดในโลก โดยทีมละ 3 คน แข่งขันแก้โจทย์อัลกอริทึมภายในเวลาที่กำหนด

รูปแบบการแข่งขัน

ทีมละ 3 คน ใช้คอมพิวเตอร์ 1 เครื่อง แก้โจทย์โปรแกรมมิ่งประมาณ 10–12 ข้อ ภายใน 5 ชั่วโมง ใช้ภาษา C, C++, Java หรือ Python ได้

คุณสมบัติผู้สมัคร

นิสิตนักศึกษาระดับปริญญาตรีหรือเทียบเท่า ที่กำลังศึกษาอยู่ในสถาบันอุดมศึกษาในประเทศไทย ตามเงื่อนไข ICPC Regional Rules

รอบการแข่งขัน

ทีมที่ผ่านรอบภาคกลาง จะได้สิทธิ์เข้าแข่งขันรอบ ICPC Thailand National Contest และอาจได้เป็นตัวแทนประเทศไทยในรอบ Regional ของ ICPC ต่อไป
